Persona 1 – New User

Role: First-time visitor / beginner user
Goal: Successfully get started without confusion
Typical Actions:

Visits the site/app for the first time

Registers or signs up

Logs in and explores basic features
Risks / Edge Cases:

Enters invalid information

Confused by unclear instructions

May abandon if things don’t work the first time


Persona 2 – Returning User

Role: Regular existing user
Goal: Complete tasks quickly and reliably
Typical Actions:

Logs in frequently

Uses core features repeatedly

Expects consistent performance
Risks / Edge Cases:

Forgetting password

Session timeouts

Unexpected errors interrupting normal workflow



Persona 3 – Power User / Admin

Role: Advanced or administrative user
Goal: Manage settings, data, or multiple users efficiently
Typical Actions:

Accesses admin tools

Edits or manages system content

Views analytics or system state
Risks / Edge Cases:

Higher impact when errors occur

Needs stronger security validation

Needs accurate permissions handling

Persona 4 – Mobile User

Role: User accessing system on phone or tablet
Goal: Complete actions smoothly on mobile
Typical Actions:

Uses site on mobile browser

Interacts with smaller UI components

Possibly on slower connection
Risks / Edge Cases:

UI layout breaking

Buttons too small / elements misaligned

Performance delays


Persona 5 – Accessibility-Focused User

Role: User with accessibility needs
Goal: Navigate and use system independently
Typical Actions:

Uses screen reader or keyboard navigation

Requires readable contrast

Depends on clear error states
Risks / Edge Cases:

Missing alt text or labels

Non-keyboard navigable UI

Poor contrast / unreadable text


Persona 6 – Unreliable Network User

Role: User on weak connection or unstable internet
Goal: Still complete tasks despite slow or dropped connection
Typical Actions:

Experiences delayed loading

Refreshes pages frequently

Attempts actions multiple times
Risks / Edge Cases:

Partial saves

Timeouts

Broken workflows
